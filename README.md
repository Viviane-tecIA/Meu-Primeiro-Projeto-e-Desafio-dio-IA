Ecossistema Financeiro CAIXA - Vibe Coding
​1. Resumo do Conceito do App
​Dashboard financeiro familiar inspirado na identidade visual da Caixa Econômica Federal. O foco é facilitar o controle de gastos para famílias grandes, com uma interface simples e intuitiva desenvolvida 100% via mobile.
Link do App: https://home-funds-buddy.lovable.app
​2. Prompt Final (PRD)
​"Crie um app de finanças mobile em React com as cores da Caixa (Azul e Laranja), incluindo dashboard com saldo de R$ 4.250,00, gráficos de pizza para gastos familiares e navegação mobile-first."
​3. Evidências das Interações
​Protótipo Inicial (Bolt.new): https://caixa-fam-lia-expens-w287.bolt.host/
​Versão Final (Lovable): https://home-funds-buddy.lovable.app
​4. Reflexão sobre o Aprendizado
​Desenvolver este projeto totalmente pelo smartphone me mostrou como a IA democratiza a tecnologia. Como mãe de 4 filhos e em transição de carreira, utilizei o Vibe Coding para superar barreiras físicas, aprendendo a estruturar prompts eficazes para gerar resultados profissionais.
